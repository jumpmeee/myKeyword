1. git 충돌
mh1 브랜치에 node_modules를 포함 시키지 않고 있다가 push 하기 위해 merge 하는 과정에서 에러 발생
=> unstaged 상태의 파일이 존재해서 에러 발생
=> git merging is not possible because you have unmerged files 에러 발생
# git status로 상태 확인 후
git commit -am "메세지" a옵션 : -a 옵션을 붙이면 변경된 파일(신규로 추가된 파일을 제외)을 검출하여 인덱스에 추가하고, 그것들을 커밋하는 동작을 명령어로 실행할 수 있습니다. 

git pull
git push origin master 
마스터 브랜치일때 하기/

git checkout mh1 으로 mh1 브랜치에서 작업 중이다가 commit 한 다음에
git checkout master로 돌아가서 git merger mh1 으로 병합 한 후 
git pull (충돌방지) 다음 git push origin master 해주기
